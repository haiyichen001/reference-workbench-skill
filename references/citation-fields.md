# 引用类型字段检查表

查证时按类型逐字段比对。加粗字段为必须匹配项。

## 期刊论文

作者(Author) / 标题(Title) / 期刊名(Journal) / 年份(Year) / 卷号(Volume) / 期号(Issue) / 页码(Pages) / DOI / ISSN

## 会议论文

作者(Author) / 标题(Title) / 会议名(Conference) / 年份(Year) / 页码(Pages) / 地点(Location) / DOI

## 书籍

作者/编者(Author/Editor) / 书名(Book Title) / 出版商(Publisher) / 年份(Year) / 版次(Edition) / 出版地(Location) / ISBN

## 书籍章节

章节作者(Chapter Author) / 章节标题(Chapter Title) / 书名(Book Title) / 编者(Editor) / 出版商(Publisher) / 年份(Year) / 页码(Pages)

## 学位论文

作者(Author) / 标题(Title) / 学位类型(MA/MS/PhD) / 授予机构(Institution) / 地点(Location) / 年份(Year) / 导师(Advisor)

## 预印本

作者(Author) / 标题(Title) / ArXiv ID或等同标识 / 年份(Year) / 版本号(Version)

## 专利

发明人(Inventor) / 专利标题(Title) / 专利号(Patent Number) / 授权日期(Date) / 专利局(Patent Office)

## 技术报告

作者/机构(Author/Institution) / 标题(Title) / 报告号(Report Number) / 发布机构(Institution) / 年份(Year)

## 报纸文章

作者(Author) / 标题(Title) / 报纸名(Newspaper) / 日期(Date) / 版面(Page/Section)

## 网页资源

作者/组织(Author/Org) / 标题(Title) / URL / 发布日期(Date) / 访问日期(Access Date)

## 标准

标准组织(Standards Body) / 标准号(Standard Number) / 标题(Title) / 年份(Year)
